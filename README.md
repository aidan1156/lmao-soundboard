Soundboard everything you say automatically, this was created during a half day hackathon where we spent the first 2 hours trying to get a sponsor's speech to text to AI model working before giving up and switching to this.

The project uses browser speech to text and a server side sentiment embedding of the user's speech to find a relevant sound to play 

run server.py with all the requirements and then visit the URL. Requirements.txt is probably not even complete and contains packages that aren't needed, this was made for a hackathon did I mention?
